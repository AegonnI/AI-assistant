Для работы модуля **recommendation_module** необходимо установить и настроить среду Ollama.

---
⚠️ Важно: Смена диска для моделей

По умолчанию Ollama устанавливает модели на диск C:. Чтобы не занимать системный диск (модели весят от 5 ГБ), следуйте этой инструкции до установки:

   1. Создайте папку для моделей на нужном диске (например: D:\AI_Project\ollama_models).
   2. Скопируйте полный путь к этой папке.
   3. Настройте переменные среды:
   * Нажмите клавишу Win и введите в поиске: "Изменение переменных среды текущего пользователя" (Environment Variables).
      * В открывшемся окне (верхний блок «Переменные среды пользователя») нажмите кнопку «Создать».
      * В поле Имя переменной впишите: OLLAMA_MODELS
      * В поле Значение переменной вставьте путь к вашей папке (например: D:\AI_Project\ollama_models).
      * Нажмите ОК во всех окнах.
   
---
📥 Установка Ollama
После настройки путей выполните установку. Откройте PowerShell и вставьте следующую команду:

```bash
irm https://ollama.com/install.ps1 | iex
```

Готово! Теперь Ollama будет использовать выбранную вами папку, а ваша видеокарта подхватится автоматически.
